简介

本文是对使用hexo和github搭建个人博客过程的记录。过程中参考了hexo官网和程序羊的视频，在这里对他们进行感谢。程序羊使用了Mac电脑进行配置，且使用cnpm使下载速度更快。与之不同的是，本文所使用的是Window电脑，且仅使用npm下载。

环境及安装后各个包的版本

PC: Windows10

终端：cmder

npm：6.14.14

hexo: 5.4.0

安装hexo过程

安装npm

npm是和node.js一同安装的包管理工具，因此我们到node.js的官网https://nodejs.org/en/download/进行下载。我这里选择了 Windows Installer (.msi) 64-bit下载。下载后双击打开跟着提示一步一步安装即可。

完成后打开终端输入npm -v即可查询npm版本号。这里我选用的终端是cmder，windows自带的cmd也是一样的，但是powershell却不成功。 最初我直接打开Cmder也不行，然后我打开了一个新的窗口，选择打开Cmder as admin输入npm就有结果了 。
